Weather Background (work in progress)

Download

Weather Background is a program that automaticaly changes your desktop background based on the weather conditions outside and the time of day (morning, day, evening, and night). Programs like this exist already; however, they are not open source, have limited coustomizability, and need an image for each weather conditon at every time of day. This limits the images you can use because they must be made specificaly for this application. Weather Background only requires four pictures for each time of day; the weather conditons are applied programaticaly.

Standard Install

Defaults

After downloading the installer exe file, double click on the file and run the installer. This installer should put the nessacary files in your program files folder along with the setup tool and image folder on your desktop. You will need to get an api key in order to access weather data. Sign up at open weather API for a free accrount and API key. Next find 4 images you want to use for your desktop background on the internet, one for morning, day, evening, and night. Store these files inside the backgrounds folder on your desktop. Next, run the setup tool that is located on your desktop. Here you can copy and paste your API key, enter the path for the backgrounds folder(which you can relocate prior to this step), your city, and the font size of the HUD in the top right hand corner. You can press the test button to ensure that it works or press set.
